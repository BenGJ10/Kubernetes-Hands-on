# Kubernetes Hands-On

This repository contains basic Kubernetes Pod manifest examples for practice and learning. It starts with simple Pod configurations and progresses to more complex setups and concepts.

## Prerequisites

- A running Kubernetes cluster (for example: Minikube, kind, Docker Desktop Kubernetes, or a cloud cluster)

- `kubectl` installed and configured

- Access to a namespace where you can create Pods
